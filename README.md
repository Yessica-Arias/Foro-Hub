ForoHub Api 

Este proyecto forma parte dell challenge de backend donde se usa un CRUD para la administracion de topicos.

Funciones:

1. Crea un nuevo topico
2. Lista topicos con paginacion
3. Ver el detalle de un topico especifico
4. Actualiza la informacion de un topico
5. Elimina un topico
6. consulta los primero 10 topicos oordenados
7. Busca topicos por curso
